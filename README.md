# 🍎 Fruit & Vegetable Market Management System

A full-stack Fruit & Vegetable Market Management System developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

This project helps manage fruits and vegetables inventory with features like adding, updating, deleting, searching, and filtering products. It also calculates total inventory value and displays low stock alerts.

---

## 🚀 Features

- Add Products
- Update Products
- Delete Products
- Search Products
- Category Filter
- Low Stock Alert
- Total Inventory Value
- Responsive UI
- MongoDB Database Integration
- REST API Support
